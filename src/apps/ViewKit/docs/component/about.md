# What is components?

This file explains components in ViewKit.

## Overview
All components in ViewKit are defined in HTML format.
These reside in the resources/components/ directory.

Also, almost all of these designs (except for unnecessary elements like text) are managed in .pen files.
An editor for these can be downloaded from pencil.dev.
